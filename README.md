# TrussC.jl

[TrussC](https://trussc.org/) Julia binding.

C++ bindings are generated by [tcxJuliaCxxWrap](https://github.com/funatsufumiya/tcxJuliaCxxWrap) (and precompiled in [TrussC_prebuilt_jll](https://github.com/funatsufumiya/TrussC_prebuilt_jll))

> [!NOTE]
> This repository is not registered in public package manager, so you need to add this via `pkg> add https://github.com/funatsufumiya/TrussC.jl.git` (and also TrussC_prebuilt_jll as `pkg> add https://github.com/funatsufumiya/TrussC_prebuilt_jll.git`)

## Usage

see https://github.com/funatsufumiya/TrussCExample.jl
